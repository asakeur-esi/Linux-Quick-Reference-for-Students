TP1 - Basic Commands

Session Management

CommandDescription
script Lab1Save terminal session to file
script -a Lab1Append to existing script file
exitExit script session

System Information

CommandDescription
whoShow logged in users and terminal ID
whoamiShow current username only
cal -3Display 3 months (previous, current, next)
dateShow current date and time

Calculator

CommandDescription
bcStart calculator
scale=2Set decimal places
quitExit calculator

Navigation

CommandDescription
pwdShow current directory
cd /absolute/pathChange directory using absolute path
cd ../relative/pathChange directory using relative path
cd ~Go to home directory
cd ..Go to parent directory
cd /Go to root directory
Note:
  • Absolute path: Begins with root / and gives all parent directories
  • Relative path: ../file.txt (depends on current location)

Directory Management

CommandDescription
mkdir dirnameCreate directory
mkdir dir1 dir2Create multiple directories
rmdir dirnameDelete empty directory
rm -r dirnameDelete non-empty directory

File Operations

CommandDescription
touch file.txtCreate empty file
touch .hiddenCreate hidden file
cat > file.txtWrite to file (overwrites), Ctrl-C to save
cat >> file.txtAppend to file, Ctrl-C to save
cat file.txtDisplay file content

Listing Files

CommandDescription
lsBasic listing
ls -lLong format (permissions, owner, size, date)
ls -iShow inode numbers
ls -aShow all files (including hidden)
ls -F /Show file types (/ for dirs, * for executables)
ls -lhHuman-readable file sizes
ls -alAll files + long format

Copy, Move, Delete

CommandDescription
cp source destCopy file
cp ../file .Copy from parent to current directory
mv old newRename file
mv file dir/Move file to directory
rm fileDelete file

TP2 - Archives, Compression, Links

TAR Archives

CommandDescription
tar -cvf backup.tar /pathCreate archive (c=create, v=verbose, f=file)
tar -tvf backup.tarList archive contents (t=list)
tar -xvf backup.tarExtract archive (x=extract)
tar -zcvf backup.tar.gz /pathCreate compressed archive (z=gzip)

Compression

CommandDescription
gzip fileCompress file (creates file.gz)
gunzip file.gzUncompress file

Links

CommandDescription
ln source hardlinkCreate hard link
ln -s target symlinkCreate symbolic (soft) link
ln -s ~ dirnameCreate symbolic link to directory
ls -li fileShow inode and link count
Links:
  • Hard link: Multiple names for same file (same inode)
  • Soft link: Shortcut to file/directory (different inode)

Wildcards

PatternMatches
*Any characters
?Single character
[13]Characters 1 or 3
[0-9]Any digit
*s*Files containing letter s
??s*Files with s as 3rd character

TP3 - Redirection and Piping

Output Redirection

CommandDescription
command > fileRedirect stdout to file (overwrite)
command >> fileRedirect stdout to file (append)
ls ~ > files.txtSave directory listing to file

Error Redirection

CommandDescription
command 2> fileRedirect stderr to file
command 2> /dev/nullDiscard error messages
command > out.txt 2>&1Redirect both stdout and stderr

Input Redirection

CommandDescription
command < fileRedirect stdin from file
sort < input.txtSort contents from file

Piping

CommandDescription
command1 | command2Send output of command1 to command2
ls -l ~ | sortList and sort
command | tee fileDisplay output and save to file
command | tee -a fileDisplay and append to file

Utility Commands

CommandDescription
head -5 fileShow first 5 lines
tail -5 fileShow last 5 lines
sort fileSort lines alphabetically/numerically
Remember:
  • > overwrites file
  • >> appends to file
  • 2> redirects errors only
  • | connects commands

Change Permissions (chmod)

Symbolic Mode:

CommandDescription
chmod +r fileAdd read permission
chmod -w fileRemove write permission
chmod +x fileAdd execute permission
chmod u+x fileAdd execute for user only
chmod g-w fileRemove write from group

Numeric Mode:

CommandResult
chmod 755 filerwxr-xr-x
chmod 644 filerw-r--r--
chmod 777 filerwxrwxrwx

Change Ownership

CommandDescription
chown user fileChange file owner
chgrp group fileChange file group
chown user:group fileChange both owner and group

Umask

CommandDescription
umaskShow current umask value
umask 022Set new umask value
Default Permissions:
  • Directories: 777 - umask
  • Files: 666 - umask
  • umask 022 ‚Üí dirs: 755, files: 644

TP5 - FIND and GREP Commands

FIND Command

Search by Name:

CommandDescription
find -name filenameSearch in current directory
find / -name filenameSearch in all directories
find -name '*.sh'Find all shell scripts

Search by Type:

CommandDescription
find . -type fFind only files
find . -type dFind only directories
find . -type lFind only symbolic links

Search by Size/Time:

CommandDescription
find -size +1kFiles larger than 1KB
find -size -4kFiles smaller than 4KB
find -mtime +2Modified more than 2 days ago

Execute Commands:

CommandDescription
find -name '*.c' -exec ls -l {} \;Run ls on all .c files
find . -type f -exec chmod 664 {} \;Change permissions

GREP Command

Basic Search:

CommandDescription
grep pattern fileSearch for pattern in file
grep -r pattern ~Search recursively
grep -n pattern fileShow line numbers
grep -i pattern fileCase insensitive

Pattern Matching:

PatternMeaning
^Beginning of line
$End of line
.Any single character
|OR (with -E)
[]Character set

TP6 - Linux Filters

SORT Command

CommandDescription
sort fileSort alphabetically
sort -r fileReverse order
sort -n fileNumeric sort
sort -u fileRemove duplicates
sort -t: -k2 fileSort by column 2, delimiter :

TR Command

CommandDescription
tr 'a' 'z' < fileReplace a with z
tr -d "'" < fileDelete apostrophes
tr '[:upper:]' '[:lower:]'Convert to lowercase
tr -s ' ' < fileSqueeze spaces

CUT Command

CommandDescription
cut -f1 -d: fileExtract field 1, delimiter :
cut -f1,3 -d: fileExtract fields 1 and 3
cut -c2-10 fileExtract characters 2-10

PASTE Command

CommandDescription
paste file1 file2Merge files side by side
paste -d: file1 file2Merge with : delimiter

Other Filters

CommandDescription
split -b 100 fileSplit into 100-byte chunks
split -l 3 fileSplit into 3-line chunks
wc fileCount lines, words, bytes
wc -l fileCount lines only
comm file1 file2Compare sorted files (3 columns)
diff file1 file2Show differences between files
cmp file1 file2Compare byte by byte

TP7 - Process Management and User Administration

Process Management

PS Command (Process Status):

CommandDescription
psShow current user processes
ps -uShow processes with details
ps -fFull format listing
ps -lLong format
ps -eShow all processes
ps auxAll processes with detailed info

Background and Foreground:

CommandDescription
command &Run command in background
fgBring background process to foreground
bgResume suspended process in background
Ctrl+ZSuspend current foreground process
jobsList background jobs

Kill Processes:

CommandDescription
kill PIDTerminate process by PID
kill -9 PIDForce kill process (SIGKILL)
kill -15 PIDGracefully terminate (SIGTERM)
killall process_nameKill all processes by name
killall -9 catForce kill all cat processes
Common Signals:
  • -9 (SIGKILL): Force terminate (cannot be ignored)
  • -15 (SIGTERM): Graceful terminate (default)
  • -STOP: Suspend process
  • -CONT: Resume process

Process Monitoring:

CommandDescription
topReal-time process monitor (press q to quit)
nohup command &Run process that continues after logout

User Administration

User Management:

CommandDescription
sudo adduser usernameCreate new user (interactive)
sudo passwd usernameSet/change user password
sudo userdel usernameDelete user
sudo userdel -r usernameDelete user and home directory

User Modification (usermod):

CommandDescription
sudo usermod -c "comment" userAdd user description
sudo usermod -d /new/home userChange home directory
sudo usermod -e 2024-12-31 userSet account expiry date
sudo usermod -g groupname userChange primary group
sudo usermod -l newname oldnameChange username/login
sudo usermod -L usernameLock user account
sudo usermod -U usernameUnlock user account

Group Management:

CommandDescription
sudo groupadd groupnameCreate new group
sudo groupdel groupnameDelete group
sudo adduser user groupnameAdd user to group
groups usernameShow user's groups
id usernameShow user ID and group IDs

System Control:

CommandDescription
sudo shutdown -P nowPower off immediately
sudo shutdown -r nowReboot immediately
sudo shutdown -h +10Shutdown in 10 minutes
sudo shutdown -cCancel scheduled shutdown
sudo rebootReboot system
User Information Files:
  • /etc/passwd: User account info
  • /etc/shadow: Encrypted passwords
  • /etc/group: Group information
